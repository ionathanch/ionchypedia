==============
i3 Keybindings
==============

These are all prefixed with `$mod`.

Notation:

* Capitalization: Shift
* ``^``: CTRL
* \*: custom keybindings

Basics
------

================= =======
Effect            Command
================= =======
Terminal          enter
Reload i3 configs C
Restart i3        R
Lock screen       ^L\*
System mode       0
================= =======
